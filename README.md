# 🖼️ Logo Image Optimizer

A simple Python-based tool to optimize logo images by converting, resizing, and compressing them automatically.

## ✨ Features
- Convert images to JPEG format
- Resize images to 200x200 pixels
- Compress images to a maximum of 50KB
- Support multiple file uploads
- Automatic download of processed files

## 📂 Supported Formats
- PNG
- JPG / JPEG
- WEBP

## 🚀 How to Use
1. Open the Google Colab link below
2. Upload your images (single or multiple files / folder)
3. Run all cells
4. Processed images will be downloaded automatically

## 🔗 Try it on Google Colab
https://colab.research.google.com/drive/1SWlLuF_C-kU_9OQEYCnaRQVF3tOPA9i3

## 📁 Output
- Single file → downloaded directly
- Multiple files → downloaded as ZIP

## 🛠️ Tech Stack
- Python
- Pillow (PIL)
- Google Colab

## 📌 Notes
- Output images are optimized for lightweight usage (≤ 50KB)
- Transparent images will be converted to white background
